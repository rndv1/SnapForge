# Журнал изменений

**Язык:** Русский | [English](CHANGELOG.en.md)

## Не выпущено

### Добавлено

- Кодекс поведения и политика безопасности для GitHub Community Standards.
- Вычитана русская документация и переведены оставшиеся английские заголовки.
- Генерация `SHA256SUMS.txt` для релизных файлов.
- Русская документация как основной вариант.
- Английские варианты документации с переключателями языка.
- Готовые архивы веб-приложения, где `snapforge.exe` запускает локальный веб-интерфейс и открывает браузер.
- Переключатель языка `RU` / `EN` в веб-интерфейсе.

### Изменено

- Создание GitHub Release теперь выполняется через GitHub CLI вместо стороннего action для релизов.
- CLI-архивы переименованы в `snapforge-cli-*`, чтобы не путать терминальный CLI и готовое веб-приложение.

## 0.1.0

Первый релиз SnapForge.

### Добавлено

- CLI-команда для генерации карточек из скриншотов.
- PNG-рендеринг через ImageSharp.
- Темы `light` и `dark`.
- Встроенные пресеты для GitHub, Open Graph, соцсетей, портфолио и презентаций.
- Пользовательские цвета фона.
- Настройка отступов.
- JSON-конфиги.
- Пакетная генерация.
- Локальный веб-интерфейс.
- GitHub Actions CI.
- Процесс релизов для готовых CLI-архивов и пакета .NET tool.
